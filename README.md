# Bible Garden — Website

Landing page for the [Bible Garden](https://github.com/Bible-Garden) app. Static HTML site with 3 language translations (EN, RU, UK).

Live: [bible.garden](https://bible.garden)

## Features

- Coverflow phone carousel with app screenshots
- Canvas firefly particle animation
- Responsive design (Tailwind CSS)
- i18n with localStorage persistence

## Stack

- HTML + Tailwind CSS (CDN)
- Google Fonts: Lora + Inter
- Vanilla JavaScript
- No build step required

## Run locally

```bash
python3 -m http.server 8080
# open http://localhost:8080
```
